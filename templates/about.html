About MediSense

MediSense is an AI-powered health recommendation system that predicts possible diseases based on user-selected symptoms. It provides personalized suggestions including medicines, diet plans, and workout routines.

🔍 What this project does:

  • ✔ Predicts diseases using Machine Learning
  • ✔ Suggests medicines
  • ✔ Provides diet recommendations
  • ✔ Gives workout advice

⚙️ Technologies Used:

  • Python & Flask
  • Machine Learning (Scikit-learn)
  • HTML, CSS, JavaScript
